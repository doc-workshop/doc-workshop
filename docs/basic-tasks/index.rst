初级任务
========

本章节将带你完成以下初级任务：

- 使用 RST 编写技术文档
- 在本地使用 Sphinx 将文档编译为 HTML，并查看效果

**交付：**

- 本阶段完成后，应自主编写一份 RST 文档，需要用到 :doc:`rst-syntax` 中介绍的多数功能。编写完成后，在 :doc:`check-rst-syntax` 页面提交。
- 本阶段结束后，应在本地编译一份 HTML 文档，其中 author 为自己小红书账号名称，主题为：alabaster。完成后，在 :doc:`check-html-output` 页面提交生成的 ``_build/html/index.html`` 文档。
